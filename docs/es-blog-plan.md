## 一、Blog 页面结构

参考 DE 站布局，ES Blog 页面结构如下：

1. **Hero 区域** — 标题 + 描述
   - H1: `Conocimiento experto para importadores hispanos`
   - 副标题: `Guías, análisis y consejos sobre fabricación OEM/ODM, certificaciones, importación desde China y tecnología de carga.`

2. **精选文章网格** — 2 列卡片网格，显示最新 4-6 篇文章
   - 每张卡片：封面图 + 分类标签 + 标题 + 摘要 + 日期 + 阅读时间

3. **所有文章** — 按分类折叠的完整文章列表（与 DE 站一致）
   - 分类标题 + 文章链接 + 发布日期

4. **快速访问** — 快捷入口卡片（Power Bank、Qi2、Importación 等）

5. **CTA 区域** — 询盘表单

## 五、作者分配策略

- **Snowy May**（Market Manager）— 侧重：tecnología, certificaciones, tendencias de mercado, guías técnicas
- **Nina Nico**（Sales Manager）— 侧重：proceso OEM/ODM, importación, control de calidad, logística, cumplimiento normativo
- 每批文章交替署名，避免连续 2 篇以上同一作者

## 六、内容本地化要点

- **术语一致性**：使用西班牙本土西班牙语（"power bank" 而非 "batería externa"、"móvil" 而非 "celular"），但兼顾 LATAM 可读性
- **本地法规**：强调 UE 2023/1542, AENOR, WEEE, GPSR 代表等西班牙市场特定要求
- **物流信息**：DDP 西班牙、IVA de importación、Amazon FBA España
- **文化参照**：使用西班牙和 LATAM 本地化案例和参照
- **价格单位**：EUR 为主，USD 为辅

## 七、封面图方案

- 首批复用 DE 站 cover 图（替换文本为西班牙语）
- 新文章需制作 ES 专用 cover，统一尺寸 800×384px
- 存储路径：`/es/blog/cover/{article-slug}.webp`

## 八、URL 结构

```
/es/blog/{article-slug}/
```

Slug 命名规则：西班牙语关键词，连字符分隔，全小写，无停用词。

示例：
- `fabricante-power-banks-china-oem`
- `certificacion-qi2-importadores`
- `importar-cargadores-china-aduanas`
